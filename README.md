# 随机展现自拍私拍的优质帖子-AI生成

一个用于展示自拍、私拍类帖子的前端展示系统，支持随机浏览帖子内容、图片轮播等功能。

## 🌐 在线访问

点击右侧page页面即可访问

## 功能特性

- 📱 **帖子展示**：随机加载并展示帖子内容
- 🖼️ **图片轮播**：支持多张图片的切换浏览，支持鼠标点击和键盘方向键操作
- 📊 **数据统计**：展示帖子的浏览量、评论数、点赞数
- 👤 **用户信息**：显示发帖用户的昵称、年龄、等级等信息
- 🎨 **精美界面**：采用现代化的卡片式设计，支持响应式布局

## 文件结构

```
huluxia/
├── index.html          # 主页面，包含前端展示逻辑
├── README.md           # 项目说明文档
├── post1/              # 示例帖子数据目录
│   └── data.json       # 示例帖子数据
└── posts2/             # 帖子数据目录
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 使用方法

### 本地运行

1. 将项目文件下载到本地
2. 使用浏览器打开 `index.html` 文件即可运行

### 功能操作

- **换一篇**：点击页面底部的"换一篇"按钮，随机加载新的帖子
- **图片切换**：
  - 鼠标点击左右导航按钮切换图片
  - 使用键盘方向键（← →）切换图片
- **图片预览**：鼠标悬停在图片上会放大预览

## 数据格式

帖子数据采用 JSON 格式存储，包含以下字段：

```json
{
    "postID": "帖子ID",
    "title": "帖子标题",
    "detail": "帖子内容",
    "images": ["图片URL1", "图片URL2"],
    "hit": 浏览量,
    "commentCount": 评论数,
    "praise": 点赞数,
    "user": {
        "nick": "用户昵称",
        "age": 年龄,
        "level": 用户等级
    }
}
```

## 数据说明

⚠️ **重要声明**：本项目中的所有帖子数据均为 AI 随机生成，仅用于演示目的。数据内容不代表真实社区的实际帖子，如有雷同，纯属巧合。

## 注意事项

- 确保 `posts2/` 目录下存在 `index.json` 文件和对应的帖子数据文件
- 图片文件需放置在 `images/downloaded_posts_images/` 目录下
